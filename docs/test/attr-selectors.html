属性セレクターのテスト

  1. [lang] 属性を持つ

    lang, lang="", lang="en", lang="en-US", lang="multi en en-US"

  2. [lang="en"] 完全一致

    lang, lang="", lang="en", lang="en-US", lang="multi en en-US"

  3. [lang~="en"] 空白区切りの語のリストの中で完全に一致するものがある

    lang, lang="", lang="en", lang="en-US", lang="multi en en-US", lang="en en-US"

  4. [lang|="en"] lang という名前の属性の値が正確に en と一致するか、 en で始まり直後にハイフン (- (U+002D)) が続く要素を表します。

    lang, lang="", lang="en", lang="en-US", lang="multi en en-US"

  5. [lang^=en] valude で始まる

    lang, lang="", lang="en", lang="en-US", lang="multi en en-US"

  6. [lang$=en] valude で終わる

    lang, lang="", lang="en", lang="en-US", lang="multi en en-US"

  7. [lang*=en] en を含む

    lang, lang="", lang="en", lang="en-US", lang="multi en en-US"

独自に定義した属性でテスト

  1. [attr] 属性を持つ

    attr, attr="", attr="value", attr="value-1", attr="multi value value-1"

  2. [attr="value"] 完全一致

    attr, attr="", attr="value", attr="value-1", attr="multi value value-1"

  3. [attr~="value"] 空白区切りの語のリストの中で完全に一致するものがある

    attr, attr="", attr="value", attr="value-1", attr="multi value value-1", attr="value value-1"

  4. [attr|="value"] attr という名前の属性の値が正確に value と一致するか、 value で始まり直後にハイフン (- (U+002D)) が続く要素を表します。

    attr, attr="", attr="value", attr="value-1", attr="multi value value-1"

  5. [attr^=value] valude で始まる

    attr, attr="", attr="value", attr="value-1", attr="multi value value-1"

  6. [attr$=value] valude で終わる

    attr, attr="", attr="value", attr="value-1", attr="multi value value-1"

  7. [attr*=value] value を含む

    attr, attr="", attr="value", attr="value-1", attr="multi value value-1"
